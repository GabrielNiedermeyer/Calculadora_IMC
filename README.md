## Tecnologias
* Kotlin
* Activity for screen host
* ViewGroup
    * RelativeLayout
    * LinearLayout
- Components de UI
    - TextView
    - EditText
    - Button
- Explicit Intent to open the second Activity
- FindViewById to retrieve UI components
- String resource for text concatenation
